核心优势

✨ Folda-Scan 为何与众不同 ✨

📁

本地项目浏览

安全访问并导航您的本地项目目录。Folda-Scan 利用 File System Access API，让一切在浏览器内完成。

👁️

实时文件监控

文件变化即时感知。优先使用实验性 FileSystemObserver API 高效检测，不支持时自动回退到轮询。

📜

智能 .gitignore 感知

自动识别并尊重您项目中的 .gitignore 规则，确保扫描和备份的纯净与高效。

⏱️

简易本地版本管理

轻松创建项目快照，随时恢复至先前版本。所有版本数据安全存储在本地的 .fe 文件夹中。

🤖

AI 友好报告生成

一键生成项目结构、文件内容和变更的 Markdown 报告，完美适配AI助手，或用作项目文档。

📊

可视化洞察

直接在应用内查看项目结构、文件内容、实时差异。让代码的脉络一目了然。

🛠️ 技术栈 揭秘

Folda-Scan 采用现代Web技术构建，为您带来流畅、高效的本地文件管理体验。
